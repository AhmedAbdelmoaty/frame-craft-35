# Statistics Business Game Workspace

This is the official workspace for the new statistics/business game.

The existing projects in the repository are workflow references only. They should be used to learn process, organization, asset pipelines, prompting patterns, testing habits, and skill usage.

The final style, theme, characters, world, and gameplay identity for this game have not been decided yet.

Reference projects must not be copied literally. Do not copy their visual style, genre, characters, enemies, story, combat systems, or specific game content into this new game.

Before implementation begins, this workspace should be used to define the learning goals, core loop, art direction, asset pipeline, and project decisions for the new game.
