# References

This folder stores imported projects and retained reference material.

References are workflow material only. They are not the final style, genre, characters, story, or gameplay identity of the new statistics/business game.

## Folders

* `imported-projects/` - original imported reference projects.
* `archive/internal-workspace-snapshots/` - preserved setup snapshots from project organization work.

Do not copy reference-project visuals or mechanics directly into the new game.
