What Works:

- Keeps the same main identity direction: young male data analyst, dark wavy hair, glasses, navy shirt, black trousers, black shoes.
- Right-side view is present inside the sheet rather than added as a separate pasted strip.
- The visible hand in the right-side view does not show the watch.
- Full body views include hands and feet.

Watch Before Spritesheets:
- This is a directional reference, not animation.
- Before generating walk cycles, inspect left/right wrist details again.
- Start with one animation only, preferably `walk-left`, and reject any output that only shakes the still image.
- Do not generate all directions in one pass.
