# Assets Workspace

This folder is for the new statistics/business game assets.

## Folders

* `references/` - approved visual references for the new game only.
* `generated/` - AI-generated raw or working assets.
* `final/` - cleaned, normalized, game-ready assets.

Imported reference-project assets should stay under `references/imported-projects/` at the repo root unless a specific item is approved as a reference for the new game.

Generated assets are not final until they pass the asset quality checklist in `docs/checklists/ASSET_QUALITY_CHECKLIST.md`.
